# Launch Guide

## Recommended

Run:

```powershell
.\start_tincap_os.ps1
```

This starts the active Next.js frontend and opens Chrome at `http://localhost:3000`.

## Manual

```powershell
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verify

```powershell
.\scripts\verify_frontend.ps1
```
